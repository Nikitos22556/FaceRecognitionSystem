# FaceRecognitionSystem — Система распознавания лиц для контроля доступа

## Описание проекта

Локальная система распознавания лиц, разработанная в рамках дипломной работы.  
Предназначена для автоматизации контроля доступа сотрудников на предприятиях.

## Технологический стек

| Компонент | Технология |
|-----------|------------|
| Язык | C# |
| Среда разработки | Visual Studio 2022 |
| GUI | Windows Forms |
| Компьютерное зрение | OpenCvSharp (OpenCV) |
| База данных | SQL Server LocalDB |

## Алгоритмы

- **Детекция лиц:** Каскады Хаара (Haar Cascade)
- **Распознавание лиц:** LBPH (Local Binary Patterns Histograms)

## Функциональные возможности

- ✅ Захват видеопотока с веб-камеры
- ✅ Детекция лица на кадре (зелёная рамка)
- ✅ Распознавание лица и приветствие
- ✅ Регистрация новых сотрудников с эталонным фото
- ✅ Административная панель (CRUD, поиск, экспорт в CSV)
- ✅ Локальное хранение данных (без интернета)

## Системные требования

- ОС: Windows 10/11 (64-bit)
- ОЗУ: от 4 ГБ
- Веб-камера: 720p (рекомендуется)
- .NET Framework 4.7.2+
- SQL Server LocalDB
